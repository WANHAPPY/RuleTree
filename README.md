# rule-tree
一个简单的规则树结构组件
功能： 
1、对单个规则进行操作
2、对规则树进行删除，上移操作
3、可以对对应模块进行扩展

maxLevel 最大的等级层级
getRuleTree() 获取结构树数据(简单进行了空值检查，会将只有一个下潜的树优化)
